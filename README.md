# redbutton

<h1>
  레드버튼 클론코딩
<p>(23.06.18 ~ 23.06.27)</p>
</h1>
<p>
 레드버튼 홈페이지의 클론코딩입니다.
</p>
<p>
  Flex를 주로 활용했으며,
  <br>
  상단 배너는 x 버튼으로 닫을 수 있고
  <br>
  GNB에서 호버시 서브메뉴탭을 볼 수 있습니다.
  <br>
  보드게임 이미지들을 클릭하면 관련 내용들을 볼 수 있습니다.
</p>
